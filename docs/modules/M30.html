M30：語法模組依賴視覺圖建構模組（Syntax Module Dependency Visualizer）

模組說明：

本模組負責將 Sean Yang Core 的語法模組依賴關係以視覺化圖表方式呈現，使開發者與使用者可快速掌握模組間邏輯鏈結、上下游依賴關係與系統整體結構圖譜。

模組功能：

  • 自動解析 M1–M44 模組間的依賴結構
  • 生成動態與靜態依賴圖（支援 Graphviz 與 HTML Canvas）
  • 提供模組點擊跳轉功能，輔助模組導航
  • 可與 M10 模組整合，強化一致性稽核能力
  • 支援匯出 SVG、PNG 或 JSON 格式
