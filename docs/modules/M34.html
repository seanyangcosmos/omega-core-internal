M34：語義模組依賴圖生成器模組（Semantic Module Dependency Graph Generator）

模組說明：

本模組負責自動繪製所有語義模組之間的依賴結構圖，協助開發者與研究者掌握模組邏輯關係、輸入輸出流動與語義架構的組織方式。亦可視為整體系統架構的視覺化支撐。

模組功能：

  • 將 M1–M44 各模組之間的依賴關係轉為視覺化拓樸圖
  • 支援層級展開、模組群組化與邏輯路徑追蹤
  • 與 M10（模組一致性稽核）搭配使用效果最佳
  • 提供 HTML + SVG 輸出與 JSON 結構匯出格式
  • 可視為全系統「語義電路圖」的基礎模組
